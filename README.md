## 此脚本可以自动下载 DMM 的 JAV 封面
### 主要功能
- 自动模式（从文件夹获取名称）
- 手动模式（手动输入番号代码）
- 竖版封面图 (Poster)
- 横版封面图 (Thumb+Fanart)
- 全部封面图 (Thumb+Poster+Fanart)
- 可自动检测番号的格式并改正，比如ABC-987,ABC987,abc987等等。
### 自动模式
可选择文件夹自动根据文件夹名称进行下载。
注意不是根据文件名哦，是最后一级的文件夹，比如 123/456/789.mkv，是提取456的名称。
### 手动模式
手动输入番号进行下载，格式会自动纠正。
此模式会下载图片至当前文件夹的 Thumb-Poster-Fanart 目录下。
